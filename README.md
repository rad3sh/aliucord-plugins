# Aliucord plugins by rad3sh

Plugins for [Aliucord](https://github.com/Aliucord), a mod for old Discord Android.

## Installation

1. Download the plugin `.zip` by clicking the plugin download button below.
2. Move the downloaded file to the `Aliucord/plugins` folder on your device.
3. Restart Discord.

## Plugins

| Plugin               | Description                                                                                                                                                                        | Download                                                                                                                                                       |
| -------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `AudioSourceChanger` | Adds an **Audio source** option directly into Discord's Voice & Video settings UI, letting you control how your microphone is captured during calls. [↓ note](#audiosourcechanger) | [![Download](https://img.shields.io/badge/Download-.zip-blue?style=flat-square)](https://github.com/rad3sh/aliucord-plugins/raw/builds/AudioSourceChanger.zip) |

## Plugin Notes

### AudioSourceChanger

> **Heads up:** this changes how Android **pre-processes** your mic audio at the system level (noise suppression, echo cancellation, gain…) before Discord even receives it — Discord applies its own processing on top. This does not control which physical microphone is used.
